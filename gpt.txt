Project Name: EchoEarnings

Description: Transitioning from a PowerShell-based Salary Calculator to a web application. The web app allows users to calculate annual and after-tax salaries based on hourly or annual income.

Key File Descriptions:

    index.html:
        Serves as the user interface for the web application.
        Includes a form for input (hourly wage or annual salary) and a display area for results.

    style.css:
        Defines the visual styling for the web application.
        Styles include layout configurations, form element designs, and result display styling.

    calculator.js:
        Core JavaScript file for handling calculations.
        Implements functions for capturing form submissions, processing calculations, and updating the web page with results.

    inputHandler.js and outputHandler.js (Optional):
        Separate scripts for handling specific aspects of the application.

        outputHandler.js is dedicated to formatting and displaying output or messages to the user.

Transition from PowerShell to Web:

    The core functionality, initially in PowerShell scripts (Calculator.ps1, InputHandler.ps1, OutputHandler.ps1), has been translated and adapted into JavaScript, aligning with web standards.
    The web application is designed to enhance accessibility and usability, targeting a broader user base compared to the PowerShell-based tool.
